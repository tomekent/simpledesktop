## SimpleDesktop.com Wallpaper Downloader ##



## Intro ##

A simple Bash script to automate the downloading of simpledesktop wallpapers. 


### To run 
just run in the folder you want them downloaded 

`$ chmod +x simpledesktop.sh`

`$ ./simpledesktops.sh`
